# historyCell

Сниппет ивента в истории активности аккаунта

## Параметры

### localization *object*

Текстовые строки

```javascript
{
  title: 'Вход в 4game.com с использованием аккаунта Фогейма',  // Описание ивента
  date: '18.07.2017 в 21:58',                                   // Дата и время ивента
  ip: 'IP: 128.71.25.1 RU'                                      // ip ивента
}
```

### type *string*

Тип ивента
