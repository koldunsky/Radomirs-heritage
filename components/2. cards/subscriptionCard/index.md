# subscriptionCard

Сниппет подписки

## Параметры

### localization *object*

Текстовые строки

```javascript
{
  tag: 'Начальный',
  duration: '6',
  durationDescription: 'месяцев',
  price: '1555',
  currency: 'rub',
  priceDescription: 'в месяц',
  discount: '33',
  buttonText: 'Купить'  
}
```

### recommended *boolean*

Рекомендованый тариф

### isLoading *boolean*

Состояние обработки запроса по нажатию кнопки
