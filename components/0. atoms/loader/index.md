# Спиннер

## Параметры

### size *string*

Размер спиннера

#### small *default*

25×25

#### large

50×50

### theme *string*

Тема страницы, влияет на дефолтный цвет спиннера

#### dark

#### light

### secondary *boolean*

Перекрашивает спиннер во вторичный цвет темы (dark → white, light → black)
